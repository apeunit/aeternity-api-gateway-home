TESTnet gateway nodes for the Æternity Blockchain

The gateway nodes allow you to interact with the aeternity blockchain using the http interface.

To interact with the aeternity node you can use one of the official SDKs:

  • Javascript
  • Python
  • Golang
  • Elixir

...or look for other interesting projects.

Join the discussion and stay up to date with the latest news via twitter.

⚠️
The gateway nodes are provided for demo/development/testing purpose and
should not be used in a production environment.

Make sure to run your own node(s) on production.
